Subscribers can easily look up a word's definition by double clicking on a word and clicking the qtip that appears.

== Description ==

Double click on the screen to highlight a word. This creates a qTip next to the word where when clicked, a pop-up will appear with the m-w.com's definition on the word. 

Please let me know of any changes you would like to see =). Post here or post your comment at the [plugin website](http://code.tseivan.com/wordpress/?page_id=2)

== Installation ==

1. Download and upload the folder `dbl-click-dictionary-look-up` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
